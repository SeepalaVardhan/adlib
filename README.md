## Installation
### Dependencies
* Python3 
* SciPy
* NumPy
* Matplotlib
* Scikit-learn
* CVXPY
* Progress
* CVXOPT (optional as a CVXPY solver)
* Jupyter Notebook (optional for notebook demo)
